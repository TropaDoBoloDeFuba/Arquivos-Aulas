## Dart: 
A linguagem Dart é a linguagem que realiza a parte dinâmica **do aplicativo**. Uma visualização do aplicativo seria:

- Dart -> Linguagem;
- Material -> CSS e HTML;
- Flutter -> Estrutura.

Exemplo:

iniciar.dart -> Base da execução;
~~~ dart
    void main() {
        print("Executar");
    }
~~~

A estrutura é bem parecida com Java, C++, Python...
